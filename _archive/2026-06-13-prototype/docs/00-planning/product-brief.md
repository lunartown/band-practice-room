# Product Brief

## Problem

합주실 이용자는 예약 가능한 시간대를 확인하기 위해 네이버 예약, 스페이스클라우드, 자체 홈페이지 등 여러 플랫폼을 각각 방문해야 합니다.

## MVP Goal

서울 주요 합주실 10~20곳의 오늘/내일 예약 가능 시간대를 한 곳에서 조회할 수 있게 합니다.

## Initial Scope

- 네이버 예약을 사용하는 합주실 우선 지원
- 비로그인 예약 페이지에서 노출되는 방/시간대 정보 수집
- 수집 결과를 DB에 저장
- 특정 날짜의 가용 시간대 조회 API 제공

## Non-Goals

- 예약 대행
- 결제
- 회원 계정
- 운영자 관리자 페이지
- 실시간 예약 확정 보장

## Success Criteria

- 대상 네이버 예약 페이지 1개에서 날짜별 예약 가능 시간대를 안정적으로 추출
- 오늘/내일 기준 조회 API 응답 가능
- 수집 실패 시 원인을 확인할 수 있는 로그 구조 확보
