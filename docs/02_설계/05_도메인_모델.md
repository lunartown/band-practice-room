- `Required`의 `●`는 필수 필드를 의미한다.
- `Example`, `Default`는 Figma에 값이 없는 경우 `-`로 표기한다.
- 필드명, 타입, enum 값은 Figma에 작성된 표기를 그대로 따른다.
- 오탈자로 보이는 값도 이 문서에서는 임의 수정하지 않는다.

## 11. ScrapeJob

한국어 명칭은 `수집 작업`으로 둔다.

`ScrapeJob`은 앞으로 실행해야 할 수집 대상을 나타낸다. 예를 들어 "어느 합주실의 어느 출처에서 어느 날짜 범위를 수집할지"를 worker가 알기 위한 작업 단위다.

| Required | Field          | Type     | Description                                        | Example | Default |
| -------- | -------------- | -------- | -------------------------------------------------- | ------- | ------- |
| ●        | id             | long     | 수집 작업 ID PK AI                                 | -       | -       |
| ●        | studioSourceId | long     | 합주실-출처 조합 ID FK (StudioSource)              | -       | -       |
| ●        | dateFrom       | string   | 수집 시작 날짜                                     | -       | -       |
| ●        | dateTo         | string   | 수집 종료 날짜                                     | -       | -       |
| ●        | status         | enum     | 수집 작업 상태 (PENDING, RUNNING, SUCCESS, FAILED) | -       | PENDING |
|          | priority       | number   | 수집 우선순위                                      | -       | 0       |
|          | runAfter       | datetime | 이 시각 이후 실행 가능한 시각                      | -       | -       |
| ●        | attempts       | number   | 실행 시도 횟수                                     | -       | 0       |
| ●        | maxAttempts    | number   | 최대 재시도 횟수, 초과 시 FAILED로 전환            | -       | 3       |
|          | lockedAt       | datetime | 처리 시작 시각, RUNNING stuck 탐지용               | -       | -       |
|          | lastError      | string   | 마지막 실패 메시지                                 | -       | -       |
| ●        | createdAt      | datetime | 생성 시각                                          | -       | -       |
| ●        | updatedAt      | datetime | 수정 시각                                          | -       | -       |

`runAfter`는 정확히 그 시각에 실행한다는 의미가 아니라, 그 시각 이후 worker가 실행해도 되는 가장 이른 시각이다. 성공 후 다음 수집 주기를 잡거나 실패 후 재시도 지연 시간을 두는 데 사용한다.

## 12. ScrapeRun

한국어 명칭은 `수집 실행` 또는 `수집 실행 이력`으로 둔다.

`ScrapeRun`은 실제로 한 번 수행된 수집 실행 기록이다. `ScrapeJob`이 해야 할 일이라면, `ScrapeRun`은 그 일을 실제로 수행한 결과다.

| -------- | ------------ | -------- | --------------------------------------------- | ------- | ------- |
| ●        | id           | long     | 수집 실행 ID PK AI                            | -       | -       |
|          | jobId        | long     | 수집 작업 ID FK, 수동 실행 시 null 가능       | -       | -       |
| ●        | studioId     | long     | 수집 대상 합주실 ID FK                        | -       | -       |
| ●        | sourceId     | long     | 수집 대상 출처 ID FK                          | -       | -       |
| ●        | dateFrom     | string   | 실제 수집 시작 날짜                           | -       | -       |
| ●        | dateTo       | string   | 실제 수집 종료 날짜                           | -       | -       |
| ●        | status       | enum     | 수집 실행 상태 (SUCCESS, FAILED, PARTIAL)     | -       | -       |
| ●        | startedAt    | datetime | 실행 시작 시각                                | -       | -       |
|          | finishedAt   | datetime | 실행 종료 시각                                | -       | -       |
|          | roomsFound   | number   | DB에 저장 또는 갱신한 방 수                   | -       | -       |
|          | slotsFound   | number   | DB에 저장 또는 갱신한 슬롯 수                 | -       | -       |
|          | errorKind    | enum     | 실패 종류 (TIMEOUT, AUTH_FAILED, PARSE_FAILED) | -       | -       |
|          | errorMessage | string   | 실패 메시지                                   | -       | -       |

`status = PARTIAL`은 날짜 범위 내 일부 날짜 또는 일부 방의 수집만 성공한 경우다. Job은 PARTIAL을 성공으로 간주하지 않으며, 다음 `runAfter` 시각에 재시도한다.

## 13. Freshness

`Freshness`는 별도 저장 엔티티가 아니라 API 응답에서 계산하는 도메인 상태다. MVP에서는 `Slot.scrapedAt`과 최근 `ScrapeRun.status`를 기준으로 계산한다.

| 상태    | 의미                                      |
| ------- | ----------------------------------------- |
| fresh   | 최근 수집 성공 후 기준 시간 이내          |
| stale   | 마지막 성공 데이터는 있지만 기준 시간 초과 |
| failed  | 최근 수집 실패                            |
| unknown | 아직 수집 전이거나 판단할 데이터 없음      |
