# 디자인 계획

> 작성일: 2026-06-13 (갱신: 2026-06-22)
> 상태: 완료 — 구조안 채택 후 실제 웹 구현으로 이어짐

## 1. 작업 순서

1. 디자인 원칙 정리
2. 화면 범위 확정
3. 화면별 역할과 진입 동선 정리
4. 사용자 상황별 여정 작성
5. 레퍼런스 수집 및 분석
6. 구조안 작성
7. 구조안 선택

## 2. 현재 위치

작업 순서 1~7을 모두 마쳤다. 단, 5번(레퍼런스 수집·분석)은 별도 경쟁사 UI 분석 대신 Claude Design 시안을 기준으로 진행했다([06_레퍼런스_분석](06_레퍼런스_분석.md) 참고).

채택한 구조안([08_구조안_선택](08_구조안_선택.md))과 시간검색 UI 결정([09_시간검색_UI_설계결정](09_시간검색_UI_설계결정.md))은 `src/web`에 실제로 구현·배포되었다. 이후 디자인 작업은 별도 문서가 아니라 코드 위에서의 반복 개선(칩·버튼 크기·리뷰 뱃지 등)으로 진행한다.
